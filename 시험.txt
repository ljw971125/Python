HDFS(하둡 분산 파일 시스템)
구글의 분산 저장 시스템에 대한 논문을 참고하여 만들어진 NDFS를 개선하여 만들어진 분산저장시스템

네트워크, I/O, RMI, Thread
원격지에 저장된 빅데이터를 병렬 분석하기 위해서 필요한 기술 

mapReduce
네트워크, I/O, RMI, Thread 기술을 몰라도 빅데이터를 병렬처리하는 어플리케이션을 만들 수 있는 자바 기반 프레임워크

mapReduce는 데이터 관리와 리소스 관리도 한다.

YARN
mapReduce의 부담을 덜어주기 위해 만들어진 도구로 이를 도입할 경우 mapReduce는 데이터 관리만 하고 해당 도구가 리소스 관리를 대신함으로써 mapReduce의 성능을 향상시키게 한다.

hadoop.apache.org
하둡의 사이트 주소

scribe
기업이 가지고 있는 비정형의 데이터를 수집 및 저장하기 위해 사용하는 도구

sqoop
기업이 가지고 있는 정형 데이터를 수집 및 저장하기 위해 사용하는 도구

java
mapReduce를 이용하여 분석을 할 경우 선행되어야 하는 컴퓨터 프로그래밍 언어

hive
mapReduce를 대체하는 sql기반의 도구

spark
mapReduce를 대체하는 도구로 배치, 실시간, 다양한 언어 사용 가능이라는 장점을 가지고 있는 도구

아파치 spark를 사용했을 때의 장점
배치 분석 가능, 실시간 분석 가능, 다양한 언어 사용 가능

oozie의 장점
직관적이고 쉬운 배치 처리 가능

하둡은 복수로 이루어진 다수의 노드와 다수의 프로세스를 관리할 관리도구가 필요하다.
관리 도구의 종류
Ganjlia, Nagios, Hue, chef

zookeeper
하둡의 시스템 안정을 위해서는 HA(고가용성)을 보장해줘야 한다. 이를 위해 secondary name node를 적용하는데, 이를 위해 최근에 가장 많이 사용하는 도구

하둡은 OLAP이다

프로젝트 수행 단계
단위테스트 -> 통합테스트 -> 시스템 테스트 -> 인수 테스트


시스템 테스트
프로젝트 수행 단계에 따른 테스트 분류 중 기능/비기능에 따른 별도의 테스트

테스트 케이스
확인해야 하는 특정 기능이나 모듈, 프로그램 수행 경로를 실행하거나 요구 사항에 준수하는지를 화긴하기 위한 테스트 과정과 이때 사용되는 입력값, 실행 조건, 그리고 예상된 결과를 정의

테스트 케이스의 필수 구성 항목
입력값, 예상 결과값(기대 결과), 테스트 절차, 테스트 단계

테스트 시나리오
일련의 테스트 절차를 명세화한 것으로 테스트 케이스의 집합으로 구성되었다.

확인 테스팅
결함이 발견되고 수정된 후, 결함이 성공적으로 제거되었는지를 확인하기 위해 테스트를 다시 수행하는 테스팅

회귀 테스팅
이미 테스트가 완료된 프로그램이 여러 이유로 수정되었을 때, 이에 대한 결함을 찾기 위해 다시 수행하는 테스팅

selenium
테스트 자동화 도구로 웹크롤링 도구로도 사용되고 있는 도구

데이터 웨어하우스
전사적으로 분산되어 있는 데이터를 구조적으로 표준화 하여 통합한 후 사용자의 의사결정에 도움주는 데이터베이스


데이터 마트
데이터 웨어하우스에서 특정 조직이 필요로 하는 특정 기능의 데이터
